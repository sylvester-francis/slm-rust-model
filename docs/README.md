## CLI Reference

```bash
python slm.py --help          # Show all commands
python slm.py info             # System information
python slm.py pipeline         # Run complete pipeline
python slm.py collect          # Generate synthetic data
python slm.py preprocess       # Merge and format datasets
python slm.py train            # Train with QLoRA
python slm.py evaluate         # Evaluate model
python slm.py convert          # Export to GGUF
python slm.py upload           # Upload to HuggingFace
python slm.py deploy           # Deploy to Ollama
```
